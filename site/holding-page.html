GlassCherry CMS

An unopinionated content management system which helps the author without hurting the developer

How does it work?

  • Total control for developers
  • Simplicity for authors

Write your code as normal

Use your usual dev tools to create your HTML. We don't care about which technologies you use.

Identify elements to be cherry-picked by the CMS

Mark elements which content authors will be able to edit using data attributes.

Let GlassCherry generate an admin site for you

Send us your annotated site source, and we'll create the admin site for your authors to use.

Edit and publish

Your content authors use the simple admin site that we host for you. Their changes create a site for you to deploy however you like.

Deploy anywhere

You updated site source can fit back into your deployment process or get published directly. We can push the changes to your git repository, to Amazon S3, to GitHub Pages, or to SFTP.
